# CYBERDEV // MAGDIEL SYSTEM

## Descrição

Este é um projeto de portfólio pessoal gamificado com tema cyberpunk, criado para um desenvolvedor Full Stack. A interface é inspirada em elementos de jogos de ficção científica, como *Cyberpunk 2077*, e apresenta uma estética hacker, com efeitos de neon, glitch e HUD (Heads-Up Display).

O projeto foi desenvolvido utilizando apenas HTML5, CSS3 e JavaScript puro, com foco em criar uma experiência de usuário imersiva e visualmente impactante.

## Estrutura de Arquivos

- `index.html`: Estrutura principal da landing page.
- `style.css`: Estilização completa, incluindo animações e efeitos visuais.
- `script.js`: Lógica de interatividade, animações e funcionalidades da página.

## Funcionalidades

- **Tela de Loading:** Animação de boot de sistema para imersão inicial.
- **Dashboard Interativo:** Painel principal com acesso ao sistema.
- **Perfil Gamificado:** Exibição de habilidades e nível do desenvolvedor.
- **Projetos como Missões:** Cards interativos que abrem modais com detalhes dos projetos.
- **Seção de Lore:** Contextualização do universo cyberpunk.
- **Sistema de XP:** Barra de progresso e sistema de níveis para engajamento.
- **Formulário de Contato Hacker:** Interface de terminal para envio de mensagens.
- **Responsividade:** Adaptado para visualização em desktops e dispositivos móveis.

## Como Executar

Basta abrir o arquivo `index.html` em um navegador web moderno. Nenhuma dependência ou build é necessária.

## Futuras Melhorias

O projeto está preparado para futuras integrações com um backend (como Flask ou FastAPI) para tornar o formulário de contato e outras funcionalidades dinâmicas.
