# PORTFÓLIO // CYBERDEV SYSTEM

## Descrição

Este é um projeto de portfólio pessoal gamificado com tema cyberpunk / RetroWave, criado por mim. A interface é inspirada em elementos de ficção científica, como *Cyberpunk 2077*, e apresenta uma estética focada em efeitos de neon, glitch e HUD (Heads-Up Display).

Originalmente concebido em HTML puro, o projeto foi evoluído para uma estrutura moderna utilizando **Vite** e **Tailwind CSS v4**, garantindo performance máxima e um deploy otimizado no Vercel.

## Estrutura de Arquivos

- `index.html`: Estrutura principal da landing page (operando como módulo Vite).
- `style.css`: Estilização completa utilizando **Tailwind CSS v4** e animações customizadas.
- `script.js`: Lógica de interatividade, sistema de idiomas (PT/EN), troca de temas e gerenciamento de modais.
- `vite.config.ts`: Motor de build e configuração das variáveis de ambiente (Gemini API).

## Funcionalidades

- **Tela de Loading:** Animação de boot de sistema para imersão inicial.
- **Dashboard Interativo:** Painel principal com acesso ao sistema.
- **Perfil Gamificado:** Exibição de habilidades e nível do desenvolvedor.
- **Projetos como Missões:** Cards interativos com efeito 3D que abrem modais com detalhes.
- **Sistema de Idiomas:** Tradução dinâmica entre Português e Inglês.
- **Hacker Terminal:** Interface de contato inspirada em linhas de comando (CMD).
- **Responsividade:** Totalmente adaptado para dispositivos móveis e desktops.

## Tecnologias Utilizadas

- **Vite:** Gerenciamento de projeto e build otimizado.
- **Tailwind CSS v4:** Estilização moderna através de utilitários.
- **TypeScript:** Configuração de ambiente e ferramentas de build.
- **Node.js:** Ambiente necessário para execução local e gestão de pacotes.
- **Vercel:** Hospedagem e deploy contínuo (CI/CD).
